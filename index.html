LOOK I'M THE <h1> TAG!

This coding cheatsheet should help me by giving me a resource of notes to circle back to when I'm stuck. The 'p' tag is what makes up this part of the content and it is wrapped in greater/less than signs

This <h2> tag is the title for the unorganized list of html tags below:

  • h1-h6: headers by size
  • div: creates a divided section to work within
  • p: creates a paragraph section for content
  • em: puts text in italics
  • strong: puts text in bold
  • br: makes a line break in the content
  • ul: makes this list
  • li: is these
  • ol: makes a numbered list
  • span: sets something apart to be styled indepently
  • img: image, requires a src and is self closing
  • alt: what is used when an image fails to load or someone uses a screen reader
  • video: yep, don't forget to add controls
  • body: is the main section of a website and is what this information is contained within
  • a: is anchor and it links an outside source to whatever it is anchored to
  • href: hyperlink reference or the address of the file
  • !DOCTYPE html: goes at the top of your html file
  • link: links your stylesheet. Needs rel= and href=
